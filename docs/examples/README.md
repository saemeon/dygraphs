# Example Gallery

A set of runnable Python scripts that double as the project's
"feature tour". Each example is a standalone `plot_*.py` file —
click through to see the source, the rendered chart, and a download
link.

Start with **Basic line chart** for a minimal working example, then
move to **Styling & error bands** for customisation, **Overlays** for
annotations / shadings / range selectors, and **Interactive Dash** for
the callback-driven data-update pattern.

Each script runs in isolation (no gallery-specific imports), so you
can copy-paste any one of them into your own project and have a
working chart.
